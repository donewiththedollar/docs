## üìä Performance Metrics

Real-time trading metrics displayed across all dashboards:

### Core Metrics
- **Total P&L** - All-time profit/loss with percentage change
- **Active Bots** - Number of running bot instances
- **Win Rate** - Percentage of profitable trades
- **Total Trades** - Cumulative executed orders
- **Account Balance** - Current USDT balance
- **Unrealized P&L** - Open position profit/loss
- **Today's P&L** - 24-hour performance snapshot

### Visual Indicators
- **Trend Charts** - Mini sparklines showing performance direction
- **Color-Coded Status** - Green (profit), Red (loss), Blue (neutral)
- **Live Updates** - Auto-refresh every 30 seconds

<Warning>Metrics update in real-time but may have a slight delay during high volatility.</Warning>

## ü§ñ Bot Management

### Bot Instance Cards

Each active bot displays:
- **Status Badge** - Running (green pulse) or Stopped (red)
- **Strategy Type** - Vortex DCA, Geometric Long Only, XGrid
- **Trading Symbols** - Active trading pairs (e.g., BTC/USDT, ETH/USDT)
- **P&L Display** - Current profit/loss for this instance
- **Trade Count** - Number of completed trades
- **Uptime** - How long the bot has been running

### Quick Actions
- **Start/Stop** - One-click bot control
- **Settings** - Jump to configuration editor
- **Delete** - Remove stopped bot instances

<Tip>You can run multiple bots simultaneously on different symbols or exchanges!</Tip>

## üîê Exchange Credentials Management

Securely store and manage API credentials for each exchange:

### Features:
- **Encrypted Storage** - Credentials secured with industry-standard encryption
- **Per-Exchange Setup** - Separate credentials for Bybit, BloFin, HTX
- **API Key Masking** - Only last 4 characters visible
- **Easy Editing** - Update credentials without bot restart
- **Delete Protection** - Confirmation required before removal

### Required Fields:
- **Bybit**: API Key + Secret
- **BloFin**: API Key + Secret + Password
- **HTX**: API Key + Secret

<Warning>Never share your API credentials. Disable withdrawal permissions for maximum security.</Warning>
